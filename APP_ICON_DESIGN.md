# QuickBites App Icon Design

## Icon Concept

**Modern, Clean, Professional**

### Design Elements:
- **Primary Color**: Gradient from #FF6B6B (coral red) to #FF8E53 (orange)
- **Shape**: Rounded square (modern Android adaptive icon style)
- **Main Symbol**: Fork and knife crossed on a white plate
- **Accent**: Lightning bolt in the center (representing "Quick")
- **Monogram**: "QB" at the top in bold white letters

### Color Palette:
- Background: Coral to Orange gradient (#FF6B6B â†’ #FF8E53)
- Plate: White (#FFFFFF)
- Utensils: Matching coral red (#FF6B6B)
- Lightning: Golden yellow (#FFD93D)

### Design Notes:
The icon communicates:
1. **Food focus** - Fork, knife, and plate imagery
2. **Speed** - Lightning bolt for "Quick"
3. **Modern aesthetics** - Clean lines, gradient background
4. **Professional** - Balanced composition, appropriate for portfolio

### To Generate Icon:
Use any vector graphics tool (Figma, Adobe Illustrator, Canva) or online icon generator to create:
- 512x512px main icon
- Android adaptive icon layers (foreground + background)
- Various densities (mdpi, hdpi, xhdpi, xxhdpi, xxxhdpi)
